## 2. 卡片（分析結果）區塊

每張卡片顯示：

- **食物名稱**
- **預估卡路里**（大字 + kcal）
- **分析摘要**（2~3 行）
- **健康建議**（icon + 文字）
- **分析時間**（相對時間 + tooltip 顯示完整）

## 3. 配色與排版

- 主色：`#6a11cb`（紫）
- 副色：`#2575fc`（藍）
- 警示：`#ff6b6b`
- 成功：`#22c55e`
- 卡片：白底、`border-radius: 1rem`、`box-shadow: 0 6px 24px rgba(0,0,0,.06)`。
- 字型：`"Inter", "Noto Sans TC", system-ui`。

## 4. 圖表規範（Chart.js）

- 卡路里趨勢：`type: 'line'`，平滑、塗底色 alpha 0.15。
- 種類分佈：`type: 'doughnut'`，cutout 65%。
- KPI 卡：純 HTML，不用圖表。
