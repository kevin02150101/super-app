How it works

Start spawns the project's dev server as a detached subprocess on this machine. Logs are written to hcas-hub/logs/<tool>.log.

Status is polled every 3 seconds via /toolbox/status. Python tools prefer their own .venv if present; the VibeSpec generator requires npm install to have been run.
